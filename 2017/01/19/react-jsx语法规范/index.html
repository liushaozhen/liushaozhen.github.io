react/jsx语法规范

发表于 2017-01-19   |   分类于 react

命名规范

  • 组件命名已大写开头

组件声明

  • return的根元素只能是一个
  • 尽量不使用 displayName，通过引用来命名

属性

  • 在组件中，使用className代替class
  • 在组件中，使用htmlFor代替for
  • 自定义属性需要加data-前缀，否则react不会渲染
  • aria-可以正常使用

样式

可以在组件中定义样式对象，然后直接使用

其它

  • 多行的JSX使用 () 包裹，有组件嵌套时使用多行模式
  • 单行的JSX 省略 ()
  • JSX中所有的标签必须闭合
