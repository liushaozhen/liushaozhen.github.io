es5与es6 react写法的不同

  1. 引入模块
    es5写法： var React=require('react');
    es6写法： var React,{Component,PropTypes} from 'react';
  2. 导出模块
    es5写法：用define和return实现
    es6写法：export default 组件名称
  3. 组件
    1. 定义组件
      es5写法 es6写法
    2. 定义组件的属性类型和默认属性(Props)
      es5写法：写在组件内部 es6写法：写在组件外边
    3. 初始化state
      es5写法 es6写法
    4. 绑定方法
      es5写法： es6写法
  4. Mixins
    使用ES6语法来创建组件是不支持React mixins的，如果一定要使用React mixins就只能使用React.createClass方法来创建组件了。
  5. 解构赋值与属性传递
    利用es6语法中的 … 运算符进行传递 下面这种写法，则是传递所有属性的同时，用新的className值进行覆盖（{…this.props}写在前边）： 这个例子则相反，如果属性中没有包含className，则提供默认的值，而如果属性中已经包含了，则使用属性中的值（{…this.props}写在后边）
